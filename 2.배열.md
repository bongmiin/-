- 프로그램이 의도된 대로 실행되지 않는 경우
	- #을 10개 출력
```cpp
#include <stdio.h>

int main(void)
{
    for (int i = 0; i <= 10; i++)
    {
        printf("#\n");
    }
}
```

> 과제 평균 구하기

```cpp
int main()
{
	 int score1 = 72;
	 int score2 = 73;
	 int score3 = 33;
	 
	 printf("평균은 : &i\n", (score1 + score2 + score3)/3);
}
```


```cpp
int main()
{
	 int scores[3];
	 scores[0] = 72;
	 scores[1] = 73;
	 scores[2] = 33;
	 
	 printf("평균은 : &i\n", (scores[0] + scores[1] + scores[2])/3);
}
```
